# 调研证据档案

> 本文件记录每条保留来源的完整调研证据，供后续章节 skill 复用。
> `research-sources.md` 只记录来源索引；本文件记录抓取、阅读、摘录、证据点和使用边界。

## 记录要求

- 每条进入 `research-sources.md` 的有效来源，必须在本文件建立同编号证据卡。
- 网页来源需记录搜索命中信息、抓取状态、正文可读性、核心内容、可用事实和风险。
- PDF / DOCX / PPTX / XLSX / CSV 等材料需记录转换方式、页码或章节位置、可用表格/指标和引用边界。
- 受版权或网页限制不能整段保存时，不要复制长篇原文；改为保存完整结构化摘要、短摘录、页码/章节定位和链接。
- 后续章节 skill 优先读取本文件，再读取 `research-notes.md` 和 `facts-ledger.md`。
