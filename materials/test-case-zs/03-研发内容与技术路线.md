# 研发内容与技术关键

## 总体目标

开发“船体结构疲劳分析报告自动生成”系统，打通以下链路：

1. 有限元结果提取
2. 疲劳关键节点识别与分类
3. 应力分布云图自动输出
4. 船级社合规报告自动生成

## 子课题 1：基于机器学习算法的船体有限元模型特征抓取

### 子课题 1.1 数据集生成的自动化方法

- 面向三类典型节点：
  - 腹板加筋型十字节点
  - 自由边型节点
  - 焊接趾端型节点
- 关键研究内容：
  - 节点特征解构与参数化定义
  - 自动化建模与批量生成
  - 单特征模型先行，多特征组合模型后续扩展
  - 自动网格划分与统一格式导出

### 子课题 1.2 数据预处理与特征工程

- 支持 bdf / dat / nas 等格式统一转换
- 统一提取节点坐标、单元连接关系、边界条件等数据
- 构建空间索引，进行节点查重与合并
- 开展网格质量诊断
- 构建几何特征、拓扑特征、力学特征三类特征体系

### 子课题 1.3 混合式节点分类模型

- 建立三类关键节点的统一描述体系
- 采用“两层分类”：
  - 第一层：区域初筛
  - 第二层：节点精细分类
- 采用监督学习、无监督聚类、半监督标签传播相结合的混合策略
- 可选算法包括随机森林、SVM、MLP、图神经网络、K-means、DBSCAN 等

### 子课题 1.4 节点分类结果可视化

- 将分类结果映射回三维网格模型
- 对不同类型节点着色高亮
- 标注节点 ID、单元 ID、分类标签

## 子课题 2：有限元软件的自动化脚本开发

### 目标

根据节点分类结果自动定位目标区域，自动输出应力分布图像，并支持多模型批量处理。

### 子课题 2.1 节点智能定位与视角优化

- 基于分类模型返回的节点信息定位目标单元
- 结合 PCA 进行视角优化
- 根据点云几何中心、协方差矩阵和特征值分解确定主视角
- 通过包围盒尺寸自适应调整视距，确保目标区域占画面 70% 以上

### 子课题 2.2 应力分布云图输出

- 解析 op2 等结果文件
- 读取应力数据并生成应力云图
- 支持主应力显示和图像文件输出

### 子课题 2.3 模型批量化处理

- 自动定位目标节点
- 自动展示优化视角
- 自动生成应力云图
- 支持多个疲劳分析模型批量运行

## 子课题 3：基于大语言模型的报告生成研究

### 子课题 3.1 规范知识图谱构建

- 将船舶行业规范、历史疲劳分析报告、内部指南转为结构化知识
- 开展多模态解析、语义分割与标签化
- 建立规则引擎和数值约束机制，确保报告合规

### 子课题 3.2 动态模板引擎

- 设计自适应报告模板
- 支持目录索引、条件分支、动态表格
- 实现计算结果、图表与模板占位符自动绑定

### 子课题 3.3 智能体构建

- 用结构化数据驱动初稿生成
- 结合知识图谱补充上下文
- 自动执行语法纠错、语义修正、数值单位校验
- 输出最终送审报告，支持 Word / PDF 等格式

## 关键技术

- 参数化训练集生成
- 混合式节点分类模型
- PCA 视角优化与动态视距调整
- 应力云图自动生成
- 规范知识图谱与规则引擎
- 动态模板引擎
- 多智能体协作式报告生成

## 创新点

- 有限元分析、机器学习和大语言模型三类技术的深度融合
- 以参数化建模解决疲劳节点数据稀缺问题
- 以知识图谱和规则约束确保送审报告合规
- 打通“数据处理 -> 机器学习分析 -> 文档生成”的端到端链路
